=====[ ФАБРИКА СОБЫТИЙ ]=====

Описание:
	Утилита для манипуляции составом событий на карте с сохранением изменений. Позволяет добавлять события из шаблона, удалять события и заменять события шаблонными.

Локация: 
	Amphicore::EventFactory

Интерфейс:
    create(map_id, name, x, y)
	Добавляет событие из шаблона
		map_id	(целое)		идентификатор карты
		name	(строка)	имя шаблонного события
		x		(целое)		координата икс
		y		(целое)		координата игрик
    
    erase(map_id, event_id)
	Стирает событие
		map_id	(целое)		идентификатор карты
		event_id(целое)		идентификатор события
    
    replace(map_id, event_id, name, x, y)
	Заменяет событие шаблоном
		map_id	(целое)		идентификатор карты
		event_id(целое)		идентификатор заменяемого события
		name	(строка)	имя шаблонного события
		x		(целое)		координата икс
		y		(целое)		координата игрик
	
Настройки:
	EVENT_FACTORY_TEMPLATES = [целое, ...]
		целое - идентификатов карт, которые используются в качестве хранилища событий шаблонов
